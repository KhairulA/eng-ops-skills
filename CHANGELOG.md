# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-09
### Added
- **Monorepo Architecture**: Pivoted from a single Outline skill to a Multi-Vertical EngOps suite.
- **Verticals**: Initial production support for **Outline Ops**, **Jira EngOps**, **GitHub EngOps**, and **GitLab EngOps**.
- **Horizontals**: Established core shared logic for **Security (PII Guard)**, **Logging**, **Auth**, and **Governance**.
- **Multi-Skill Playbooks**: Introduced cross-vertical workflows like the "EngOps Triad."
- **Developer Experience**: Rigorous validation script (`validate-skills.sh`), PR templates, and mock API samples.
- **Platform Support**: Native support for **Paperclip AI** and **Claude Code**.

### Changed
- Renamed `outline-wiki` to `outline-ops` for naming consistency across the suite.
- Overhauled root documentation for enterprise-grade compliance.
